线性基入门

  2021-12-22   笔记   数据结构 数学 Word Count: 694 words Read Time: 4 minutes

简介

线性基是一个一般用于处理集合中异或相关的问题，其时间复杂度和空间复杂度都是 $\operatorname{O}(\log a)$ 级别，$a$ 为值域。

线性基最常用的做法是兹瓷以下三个操作：

  • 向集合内插入一个数。
  • 查询集合内若干数能异或得到的最大值。
  • 查询一个数是否能被集合内若干数异或得到。

当然它还有很多副效果，比如还可以用来……

  • 查询集合内若干数能异或得到的最小值。
  • 查询集合内若干数能异或得到的第 $k$ 小值。
  • 查询集合内若干数能异或得到的第 $k$ 大值。

等等，你可以尽情扩展，只要你能做到。

本文章 $\LaTeX$ 完全按照数学标准进行，例如信息学的赋值语句不会使用 $a=b$ 表示而会表示为 $a\leftarrow b$。如有造成阅读困难，那是你自己的事，不要找我。

若无特殊注明，一个数的某一位默认为二进制下而非十进制下，没有特殊标明和特判就不考虑 $0$。

线性基基本操作

从一个例题引入已经是典中典了吧。

例 1：维护一个集合，支持两种操作：

  1. 插入一个数。
  2. 查询集合内若干数能异或得到的最大值。

$n\leqslant 10^6,1\leqslant a_i\leqslant 2^{50}$

你发现这两个操作就是前面说的基本操作的前两个……

暴力做法非常显然，复杂度是 $2^n$ 的，没必要多说。直接开正题。

线性基基本原理

线性基只需要用数组实现，记为 ${p}$。其中 $p_i$ 的最高位的 $1$ 在第 $i$ 位。

用线性基维护一个数组，我们只需要考虑能让原数组 ${a}$ 可以通过 ${p}$ 经过异或得到即可。

求线性基的时候，先考虑插入一个数 $x$。我们优先找到 $x$ 的 $1$ 的最高位（设为 $i$），如果线性基中没有对应的位，则把 $x$ 插入到线性基内，令 $p_i\leftarrow x$。

如果不是的时候，考虑如何插入。这里要从线性基维护的东西入手，因为维护线性基需要满足 ${p}$ 的元素可以异或得到 ${a}$ 的元素，从插入一个数线性基的影响的角度看，如果插入的数已经可以被线性基中的元素表示了。也就是说 $x$ 已经可以表示为若干个 $p_i$ 异或得到了，还需要更新线性基吗？

所以，从 $x$ 完全对线性基没有贡献这一特性考虑，如果 $x=p_{k_1}\oplus p_{k_2}\oplus\cdots\oplus p_{k_m}$，线性基没有变化。因此来推共性，如果最高位 $i$ 已经有数字占用了，就令 $x\leftarrow x\oplus p_i$，然后继续考虑新的 $x$ 的最高位。因为 $p_i$ 的第 $i$ 位为 $1$，$x$ 的最高位为 $i$，所以显然更新后 $x$ 的最高位小于 $i$。

如果一直更新 $x$ 直到 $x=0$ 了，都没有加入线性基，正好就说明 $x$ 已经可以被线性基内元素异或得到了，本来就不应该插入到线性基内（即对线性基造成影响），与前面对特性的推理符合。

按照这种方式插入，不难发现线性基内的数 ${p}$ 就可以表示原数组了，而且下标已经按照最高位排好序了，后续很多操作都会变得很方便，很多一部分操作都是按照低位到高位或者高位到低位顺序处理的，一些操作还带有明显的贪心性质。

不难得到线性基的数是无法异或出 $0$ 的，因此如果题目中可以有 $0$，需要特判一下。

这里提出的构造方法是不严谨的，它不满足一些线性基有的性质，对于有的题目如此构造将会无法用线性基解决，但是对于很多题目，这样直接构造即可。而且这种方法好写，写出 bug 率低。后文会指出线性基的线性代数意义和其用高斯消元构造的严谨构造法。

插入

插入的原理在前面已经有描述，不再重复。

inline bool insert(int x){
    if(x==0){
        zero=1; // 有 0 特判，后文会提为什么
        return 1;
    }
    for(register int i=n;i>=0;--i) // 保证先枚举到的是最高位
        if(x&(1LL<<i)){ // 枚举 x 的是 1 的位
            if(!p[i]){ // 线性基内没有本元素
                ++cnt,p[i]=x; // 线性基内插入，cnt 维护线性基内元素个数
                return 1; // 成功插入
            }
            x^=p[i];
        }
    return 0; // 插入失败，原值已经可以被线性基的元素表示
}

这么简单的代码几乎每一行都有注释，简直是保姆级教学了。

查询异或最大值

因为已经按位排好了，那么高位大的数值更大，所以按照高位往低位贪心地选即可。

inline int askmax(){
    int res=0;
    for(register int i=n;i>=0;--i)
        res=max(res,res^p[i]);
    return res;
}

查询一个数能否异或得到

根据前面插入的基本原理，如果这个数本身就可以被表示为线性基若干数的异或，则对线性基无影响。

反过来也成立，如果一个数插入线性基时无影响，说明它已经可以被表示为线性基若干个数的异或了。

回到线性基的性质，线性基 ${p}$ 可以表示所有原集合 ${a}$ 的元素，可以认为这两个集合在选取若干数异或的意义下相同，因此如果这个数不能被 ${p}$ 异或得到，那么也不能被 ${a}$ 异或得到。

因此和插入差不多，不需要真的插入，直接判能不能即可。

inline bool askcan(int x){// 和插入差不多
    for(register int i=n;i>=0;--i)
        if(x&(1LL<<i)){
            if(!p[i])
                return 0;
            x^=p[i];
        }
    return x==0;
}

线性基扩展

上面那三个在理解线性基的原理后都是小菜。在真正来大的之前，我们再看看如何查询异或最小值。

查询异或最小值

因为 ${p}$ 是 ${a}$ 在我们自定义的“若干异或条件下”是等价的，因此 ${a}$ 的异或最小值就是 ${p}$ 的最小元素。

值得注意的是如果 ${a}$ 中插入了 $0$，那么异或最小值应该是 $0$，这就是前面特判的原因。

inline int askmin() {
    if(zero) return 0;// 有 0 特判
    for(register int i=0;i<=n;++i) // 找最小值
        if(p[i])
            return p[i];
    return 0;// 事实上这个 return 不可能实现
}

线性基的线性代数意义

这里开始难起来了。

这里本应该有对线性代数部分知识的详解，但这种前置知识不应占太多版面，已经全部转移到我的另一个博客：浅谈线性代数。

搞定了这些前置知识之后，我们回头把线性基在线性代数上的意义过一下：

根据前面的方法，我们对线性基的认识是：通过原集合 $S$ 中的某一个最小子集 $S_1$ 组成，使得 $S_1$ 内元素相互异或得到的值域和 $S$ 得到的值域相同。

进一步，我们需要把原集合的 $n$ 个数放在二进制下，看成 $n$ 个向量，每个向量的每个元素都是 $0$ 或 $1$（即一个数的二进制位形成它代表的向量）。如果一些向量中任意一个向量能被其它向量通过异或运算得到，说明这些向量（在异或意义下）线性相关。线性基中所有的向量是线性无关的。

也就是说，线性基的线性代数意义应该是：一个向量集合的极小的线性无关子集，满足线性基中元素异或得到的值域和原向量集合相同。

（这里用到了基的性质，即可以表示出原空间内所有点，在这里空间内所有点表示的是原集合异或得到的值域。）

然后把基本的东西搞完，看看几个扩展性质：

  1. 线性基能异或出的每一个数的组成方案是唯一的。即对于任意两个不同的线性基能异或出来的数，异或出这两个数的方案一定不同且唯一。
  2. 由性质 $1$，设线性基内有 $cnt$ 个元素，那么线性基能异或得到的数的集合大小为 $2^{cnt}-1$。

下面提到的线性基的方法，全部使用其线性代数意义解决。

线性基的消元构造法

例 2：你需要维护一个集合，支持两种操作：

  1. 插入一个数。
  2. 查询集合内元素能通过选若干个数异或得到的所有数中，第 $k$ 小的数是哪个。

$n\leqslant 10^4,a_i\leqslant 2^{50}$。

我们发现线性基是按照每个二进制最高位是多少存数字的，是不是可以对 $k$ 二进值分解呢？

分解之后就可以直接二进制原理做了，但应该要保证 $p_i$ 的最高位是 $i$，而且 $p_i$ 要尽量小。

我们发现前面的构造方法是不能满足新加入的这个条件的。因此我们需要改进线性基的构造方法。

我们先假设原向量集合的基 $\mathfrak{B}$ 就是原向量集合 ${a}$，记原向量集合的张成为 $V$。然后进行如下操作：

  1. 如果 $a_1=0$，在 $\mathfrak{B}$ 中去掉 $a_1$。
  2. 如果 $a_j\in\operatorname{span}(a_1,\cdots,a_{j-1})$，其中 $j\in[2,n]$，则在 $\mathfrak{B}$ 中去掉 $a_j$。

因为每次去掉的向量都属于其前面若干向量的张成，因此它和前面的相邻线性相关，去掉就保证了 $\mathfrak{B}$ 的元素线性无关。且最后 $\mathfrak{B}$ 一定可以张成 $V$，因此 $\mathfrak{B}$ 就是原向量集合的基。

用高斯消元法可以判断一个向量能否被若干个向量张成。我们每次维护一个对角矩阵，考虑到 $a_i$ 的时候，从高到低枚举 $a_i$ 为 $1$ 的二进制位 $j$。如果 $j$ 这一行的对角线处已经是 $1$，则不能加入（即线性基这个位置已经有人选了）。因为其为对角矩阵，需要把第 $j$ 行的行向量异或 $a_i$；如果 $j$ 这一行对角线处为 $0$，就把 $a_i$ 加入到这一行，在加入一个数的时候应该要消元（维护三角性质），先用下面的数消自己，再用自己消上面的数。

我们发现除了对角线上的 $1$，别的 $1$ 能消掉就消掉，不就是让每行的数字最小吗？也就是说这样的构造方法是可以满足条件的。

举个例子，我们维护五个数的线性基：${a}={(111)_2,(001)_2,(100)_2,(011)_2,(101)_2}$。 $$ \begin{bmatrix} 0 & 0 & 0 \ 0 & 0 & 0 \ 0 & 0 & 0 \ \end{bmatrix} $$ $a_1=(111)_2\neq 0$，最高位 $1$ 在最高位，插入到第一行： $$ \begin{bmatrix} 1 & 1 & 1 \ 0 & 0 & 0 \ 0 & 0 & 0 \ \end{bmatrix} $$ $a_2=(001)_2$ 不能被 $a_1$ 张成，最高位 $1$ 在其最低为，插入到最后一行： $$ \begin{bmatrix} 1 & 1 & 1 \ 0 & 0 & 0 \ 0 & 0 & 1 \ \end{bmatrix} $$ 我们需要维护对角矩阵，用下面的数消掉上面的数，当然还有一个 $1$ 不在对角线上，反正现在消不掉，暂时不管它： $$ \begin{bmatrix} 1 & 1 & 0 \ 0 & 0 & 0 \ 0 & 0 & 1 \ \end{bmatrix} $$ $a_3=(100)_2$，最高位 $1$ 在最高位。但是最高位已经有一个数了，所以令 $a_3$ 异或最高位的数，得到 $(010)_2$，最高位 $1$ 在第二位，加入： $$ \begin{bmatrix} 1 & 1 & 0 \ 0 & 1 & 0 \ 0 & 0 & 1 \ \end{bmatrix} $$ 先看下面的行能不能消这一行的 $1$，发现不能；再看这一行的 $1$ 能不能消上面的行，发现可以消掉第一行的第二列的 $1$： $$ \begin{bmatrix} 1 & 0 & 0 \ 0 & 1 & 0 \ 0 & 0 & 1 \ \end{bmatrix} $$ 对角矩阵完全形成，剩下的数字必然无法插入了。

这样选上的 $a_i$（即 $a_1=7,a_2=1,a_3=4$ 这三个）就是 $V$ 的一个基 $\mathfrak{B}$。只不过我们还进行了高斯消元变换得到了这个矩阵。众所周知初等行变换不影响线性无关性，且初等行变换后这些向量仍然能张成原有的向量空间。

因此矩阵里的这些数，对应下来就是 $p_1=4,p_2=2,p_3=1$，也是原向量空间 $V$ 的一个基。回想前面应该满足的性质，即 $p_i$ 应该是所有满足条件中的数的最小的那个，这个矩阵就告诉我们结果了。最后线性基 $\mathfrak{B}$ 就取这个矩阵的数 ${p}$ 即可。

inline void (int *a,int m){// 构造 a 的线性基，a 大小为 m
    for(register int i=1;i<=m;++i)
        for(register int j=n;j>=0;--j)
            if(a[i]&(1LL<<j)){
                if(!p[j]){
                    p[j]=a[i];
                    for(register int k=j-1;k>=0;--k) // 先下面消这一行
                        if(p[j]&(1LL<<k))
                            p[j]^=p[k];
                    for(register int k=j+1;k<=n;++k) // 再这一行消上面
                        if(p[k]&(1LL<<j))
                            p[k]^=p[j];
                    break;
                }
                a[i]^=p[j];
            }
}

查询异或第 k 小

现在终于可以回来解决这个例题了。

我们按照消元法求出一组线性基 $\mathfrak{B}$，从小到大记为 $p_1,p_2,\cdots,p_{cnt}$，将 $k$ 二进制拆分，从低位到高位分别记为 $a_1,a_2,\cdots,a_x$，则第 $k$ 小的数为： $$ a_1\cdot p_1\oplus a_2\cdot p_2\oplus\cdots\oplus a_x\cdot p_x $$ 因为这样构造后，记 $p_i$ 为 $\mathfrak{B}$ 中最高二进制为 $i$ 的数， $p_i\oplus p_j$ 在第 $j$ 位及其更低的位的部分的数已经到达理论最小值，结合二进制原理，$p_i\oplus p_j$ 是第 $2^{i-1}+2^{j-1}$ 大异或和。

inline int getkmin(int k){
    if(zero) --k; // 特判 0
    int res=0,x=0,tmp=k;
    while(tmp){
        d[++x]=(tmp&1LL); // 求 k 的二进制拆分
        tmp>>=1;
    }
    for(register int i=0,j=1;i<=n && j<=x;++i)
        if(p[i]){
            res^=p[i]*d[j];
            ++j;
        }
    return res;
}

查询异或第 k 大

和查询第 $k$ 小差不多，类比一下即可。

当然还有不需要脑子的做法，因为我们重点讨论了如何求第 $k$ 小，又知道总共可以得到 $2^{cnt}-1$ 种不同的异或和……

inline int getkmax(int k){
    return getkmin((1LL<<cnt)-k);
}
