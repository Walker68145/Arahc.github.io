Markdown 入门

  2021-11-30   杂项 Word Count: 211 words Read Time: 1 minutes

前段时间搞了一个 $\LaTeX$ 入门，现在把 markdown 的东西也补一下。

因为 mardown 的内容比 $\LaTeX$ 的数学公式不知道少了多少，所以写起来也比较简单。下文中会现有演示，演示的下面是这一段演示的 markdown 源码，从中就可以看出格式了。

标题

因为标题会破坏文章的目录结构这里不演示。

# 这是一级标题
## 这是二级标题
### 这是三级标题
#### 这是四级标题
##### 这是五级标题
###### 这是六级标题

一级标题一般为文章的总标题，因此建议分不同部分的时候从二级标题开始划分。

字体

字体有 粗体，==高亮==，斜体，粗斜体，删除线 几个。

字体有 **粗体**，==高亮==，*斜体*，***粗斜体***，~~删除线~~ 几个。

引用块

我的知识在不断增加，你们的都在不断减少。——XeniaF

> 我的知识在不断增加，你们的都在不断减少。——XeniaF

代码

有行内代码，比如 for(register int i=1;i<=n;++i)，还有代码块：

for(register int i=1;i<=n;++i)

源码就是这样，都比较好写，代码块中三个点后面可以写 markdown 支持的代码语言。

有行内代码，比如 `for(register int i=1;i<=n;++i)`，还有代码块：

```cpp
for(register int i=1;i<=n;++i)
```

列表

列表有无序列表和有序列表：

  • 这是无序列表。
  • 想不到吧，这也是。
  • 一时间想不到这里写什么了。
+ 这是无序列表。
+ 想不到吧，这也是。
+ 一时间想不到这里写什么了。

当然无序列表也可以用一些别的符号比如 -，但是个人习惯用 +。

  1. 这是一个有序列表。
  2. 我估计你想到这也是了。
  3. 一时间又想不到这里写什么了。
1. 这是一个有序列表。
2. 我估计你想到这也是了。
3. 一时间又想不到这里写什么了。

特别的是，有序列表中，即使你的序号没有写对，它仍然会自动以 $1,2,3,\cdots$ 的序号标号。所以上面这个文字你还可以写成这样：

1. 这是一个有序列表。
1. 我估计你想到这也是了。
1. 一时间又想不到这里写什么了。

甚至还可以……

1. 这是一个有序列表。
114514. 我估计你想到这也是了。
1919810. 一时间又想不到这里写什么了。
